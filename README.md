🚦 AI-Based Intelligent Traffic Monitoring System

📌 Overview
This project uses Computer Vision and Deep Learning to monitor traffic in real time. It detects vehicles, counts them, and analyzes congestion.

🔧 Features
- Real-time vehicle detection using YOLOv8
- Vehicle counting system
- Bounding box visualization
- Simple and scalable architecture

🛠️ Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- PyTorch

📂 Project Structure
See folder structure above.

▶️ How to Run

Step 1: Clone Repository
git clone https://github.com/yourusername/AI-Traffic-Monitoring-System.git
cd AI-Traffic-Monitoring-System
Step 2: Install Dependencies
pip install -r requirements.txt
Step 3: Run Project
python src/main.py
📊 Output
Real-time detection window
Vehicle count displayed on screen
